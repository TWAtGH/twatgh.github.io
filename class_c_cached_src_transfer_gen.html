Detailed Description

Transfer generator that contains storage elements that serve as cache for source replicas.

Generates transfers based on the fixed mNumPerDay value. The main part is the selection of the best source replica. If the replica can not be transferred from a cache it will be additionally transferred there. The files are removed from the cache in dependence of how likely it is that they are used again. Assuming each subsequent access will make another access more unlikely.

Contains pairs of a ratio and file objects per access count.

The first element contains files that have been accessed only once. The second element conotains files that have been accessed twice. And so on. The ratio describes the share of files that will be used from this element. In other words, the more often a file is used, the less likely it becomes to be used again. However, the variable can be customies by the user.
