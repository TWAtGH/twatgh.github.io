Detailed Description

Singleton class that provides helper functions to load profiles and configs.

The class knows two directories. One for configuration files one for profile directories. The paths are set at program startup. Furthermore, the class provides methods to load configs or profiles by name. Loading configurations using this class will automatically resolve nested json files.
