Detailed Description

Interface for a storage resource site.

A site object represents a general data centre. This can be a grid data centre (grid site) or a cloud data centre (region). Sites have a certain geographic location and conists of multiple storage elements. A class implementing this interface must implement functionality to create, store, and access storage elements.
