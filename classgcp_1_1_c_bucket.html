Detailed Description

CStorageElement specification to represent a GCP bucket.

This class overrrides certain functionalities to allow keeping track of the introduced costs. Furthermore, functionality is added to allow the cloud processing the cost. Typically, CBucket objects are created by their owning CRegion object. The entity that creates the bucket is responsible for setting proper pricing information.

◆ GetCurStoragePrice()

auto gcp::CBucket::GetCurStoragePrice ( ) const -> long double

Helper function to get the storage price considering the currently used storage.

Returns
the price in USD per GiB per month, considering the currently used amount of storage space

The pricing details for many regions in GCP are layered. For example, storing 10 GiB of data cost x USD per GiB per month. Storing more than 10 GiB but less than 100 GiB of data cost y USD per GiB per month. This function returns the price value considering the currently used amount of storage.
