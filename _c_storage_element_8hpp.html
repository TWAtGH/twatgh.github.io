Detailed Description

Contains the CStorageElement implementation, which is based on the delegate design pattern.

Author
Tobias Wegner
Date
March 2022

The CStorageElement class delegates its functionality to an object of type IStorageElementDelegate. This allows the storage element to use different implementations while enabling sub-classes of CStorageElement to dynamically chose their base class functionality.
